==============================
安装appium
==============================

方法一
----------
npm install -g appium

npm install wd      

pip install Appium-Python-Client

方法二(建议采用)
-------
下载对应的安装包: https://bitbucket.org/appium/appium.app/downloads/

设置系统环境变量(默认安装路径)C:\Program Files (x86)\Appium\node_modules\.bin

环境检测
--------
