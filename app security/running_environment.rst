=========================== 
运行环境安全检测
=========================== 


系统环境检测
-------------

* 检测内容：

 检测在被root/越狱的Android/iOS设备上APP是否采取安全保护，如退出运行等。

* 检测方法：

 把APP安装在被root/越狱的Android/iOS设备上，查看是否有提示信息或是退出运行。

网络代理环境检测
--------------------

* 检测内容：

 检测在被HTTP代理的环境下APP是否采取安全保护，如退出运行等。

* 检测方法：

 把APP安装在被HTTP代理的环境下，查看是否有提示信息或是退出运行。
